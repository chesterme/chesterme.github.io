小记

浮动，可以把元素移到外层元素的左侧或者右侧，而浮动元素下面的内容会向上移动，围绕在浮动元素周围。 当浮动块级元素时还需要设定元素的width属性（除了图像之外，所有浮动的元素都要设置宽度），这样才能 控制块级元素占用的横占尺寸，以及还有多少空间留给下面的内容上移和围绕。

当浮动元素下面的元素设置了边框或背景色时，这些边框和背景色也会跟着元素内容一起被上移到浮动元素的左侧或者右侧， 此时，可能会看到浮动元素周围有上移元素的边框或背景色，这影响美观。可以通过设置overflow属性，让上移元素的背景色或 边框遇到浮动元素时就停止显示。

当浮动元素过高时，有可能版权信息也会上浮，没有出现的页面的底部，这不符合要求。当多个浮动元素连在一起时， 也会出现问题，如果各个浮动元素的宽度不一致时，会一个一个靠在一起；如果各自高度不一致时，就会出混乱的局面。 此时，可以使用clear属性清除哪个方向的浮动。
