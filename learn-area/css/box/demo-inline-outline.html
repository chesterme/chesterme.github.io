小记

当涉及到内外边距时，浏览器会使用不同的方式处理行内盒子，可以为行内元素添加左右内外边距，但不可以使用上下内外边距增加行内元素的高度。

有时候，需要让行内元素表现的像块级元素，或者反过来。例如，无序列表的项目都是块级元素，在页面导航中需要把连接横向显示，这时候就需要改变 列表项默认的显示方式。使用display属性，就可以让块级元素表现得像行内元素或者反过来。甚至还能让元素既像块级元素，又像行内元素。
