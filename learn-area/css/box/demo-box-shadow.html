小记

box-shadow

横向偏移
向元素的右边或左边投影，正值投向右边，负值投向左边
纵向偏移
向元素的上边或下边偏移，正值投向盒子的下边，负值投向盒子的上边
投影的半径
决定投影的模糊度和宽度，0表示不模糊，因此投影边缘清晰可见。值越大投影越模糊，也越宽。
inset
在盒子内部绘制阴影
阴影的尺寸
指的是阴影扩展的量。如果把尺寸指定为10px，浏览器会在每个方向上把阴影扩大10px
投影的颜色
可以使用rgba颜色，因为可以控制颜色的不透明度，能实现半透明效果
