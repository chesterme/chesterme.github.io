浮动问题：
  1.如果在一个没有浮动的外层标签（例如一个div）中浮动一个或多个元素，会遇到一个问题：浮动的元素比div标签其他内容高时，会从外层元素的底部溢出。如果外层元素有背景或边框时，
  问题更加明显。
  2.如果浮动元素比外层的div标签本身高，会从容器的底部溢出；
  3.如果外层标签有背景或边框，溢出的元素甚至想不是在容器中。
  4.浮动的元素还可能碰到其他元素（包括其他浮动的元素），从而形成一种“阶梯”效果，不会整齐排列。

解决浮动问题：
  1.在外层元素的底部添加一个元素，清除浮动。只需要在包含浮动元素的div标签尾部添加一个标签，然后使用clear属性强制那个标签显示在浮动的元素下面。这个技巧会撑大外层的div标签，包括背景和边框。
  2.浮动外层元素：为外层div添加一个div容器，浮动包含浮动元素的div标签，浮动的div容器能够完全包含里面所有的浮动元素。此外，还有为浮动容器后面的元素添加clear属性，确保后续元素下坠，显示在容器下方。
  3.使用overflow：hidden。在外层容器中添加overflow：hidden；它能强制外层容器扩大，包含浮动元素。一般来说，这种效果很好，可是，容器中绝对定位的元素会消失。
  4.使用mivro clearfix方法。即
    .clear:after {
      content: "";
      display: table;
      clear: both;
    }
